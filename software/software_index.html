CONTACT 		Sage Control System Toolbox

I am in the process of building a Control System Toolbox for Sage. Sage is a free open-source mathematics software system licensed under the GPL. It combines the power of many existing open-source packages into a common Python-based interface.

Here are the modules that are / will be built:
• Controllability
• Observability
• Continuous Algebraic Ricatti Equation
• Transfer Function
• State-Space
• Bode Plot
• Nyquist Plot
• Root Locus
• Step Response
• Impulse Response
• LQR
• LQG
